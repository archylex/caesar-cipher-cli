# caesar-cipher-cli

## Parameters

- -a, --action: an action encode/decode
- -s, --shift: a shift
- -i, --input: an input filename path
- -o, --output: an output filename path

## Examples

```console
:~$ node main.js -action encode -shift 3
```

```console
:~$ node main.js -a decode -s 3 -i ./file1 -o ./file2
```
